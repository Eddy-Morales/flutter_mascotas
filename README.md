# Sistema de Gestión de Campañas de Vacunación

## Descripción

Este proyecto permite gestionar campañas de vacunación mediante diferentes perfiles de usuario, cada uno con funcionalidades específicas dentro del sistema.

---

# Roles del sistema

## 👨‍💼 Coordinador de Campaña

El Coordinador de Campaña es responsable de administrar la campaña, gestionar brigadas, supervisar el avance y consultar la información general.
